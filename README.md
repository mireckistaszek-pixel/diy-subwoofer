DIY 15" Subwoofer – PC Bass Reflex Build
📌 Project Description

This project is a DIY 15" subwoofer designed for use with a computer. The goal is to create a powerful bass system on a budget (~$400) with a custom-built bass reflex enclosure (ported box).

The enclosure was designed using Subbox.pro, built from MDF, and equipped with a high-quality 15" subwoofer and a mono amplifier. The system connects directly to a PC via 3.5mm jack to RCA, making it plug-and-play.

📦 Enclosure Design
Type: Bass Reflex (slot port – side channel)
Material: MDF 18mm
Example Dimensions:
Width: ~24 in (61 cm)
Height: ~16.5 in (42 cm)
Depth: ~10–12 in (25–30 cm)

Construction:

Screws + wood glue + silicone for a fully sealed box.
Slot port design enhances low-frequency response.
🔊 How It Works
Audio signal from the PC goes to the mono amplifier (Fosi Audio M03).
The amplifier drives the 15" subwoofer inside the bass reflex enclosure.
The enclosure and porting enhance the bass and overall efficiency.

Settings Example:

Low-pass filter: 80 Hz
Gain: 30–50%
Bass boost: 0 dB
